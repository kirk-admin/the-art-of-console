1. Basic Color Styling

Change text colors using the %c placeholder

console.log("%cRed text", "color: red;"); console.log("%cBlue text", "color: blue;"); console.log("%cGreen text", "color: green;"); console.log("%cPurple text", "color: purple;");

2. Font Styling

Change font size, weight, and family

console.log("%cLarge bold text", "font-size: 24px; font-weight: bold;"); console.log("%cSmall italic text", "font-size: 12px; font-style: italic;"); console.log("%cMonospace font", "font-family: monospace; font-size: 16px;");

3. Background Colors

Add background colors to your console text

console.log("%cRed background", "background: red; color: white; padding: 5px;"); console.log("%cBlue background", "background: blue; color: white; padding: 5px;"); console.log("%cYellow background", "background: yellow; color: black; padding: 5px;");

4. Gradients

Create beautiful gradient backgrounds

console.log("%cRainbow gradient", "background: linear-gradient(45deg, red, orange, yellow, green, blue, purple); color: white; padding: 10px; border-radius: 5px;"); console.log("%cBlue to purple", "background: linear-gradient(90deg, #4facfe, #00f2fe); color: white; padding: 10px; border-radius: 5px;");

5. Borders and Padding

Add borders, padding, and rounded corners

console.log("%cBordered text", "border: 2px solid red; padding: 10px; border-radius: 5px;"); console.log("%cDashed border", "border: 3px dashed blue; padding: 8px;"); console.log("%cRounded corners", "background: #e74c3c; color: white; padding: 10px; border-radius: 20px;");

6. Multiple Styled Elements

Style different parts of the same message

console.log("%cSuccess: %cOperation completed", "color: green; font-weight: bold;", "color: black;"); console.log("%cError: %cSomething went wrong", "color: red; font-weight: bold;", "color: black;"); console.log("%cInfo: %cThis is information", "color: blue; font-weight: bold;", "color: black;");

7. Advanced Styling

Complex styling with shadows and effects

console.log("%cShadow effect", "text-shadow: 2px 2px 4px rgba(0,0,0,0.5); font-size: 20px; font-weight: bold;"); console.log("%cGlow effect", "text-shadow: 0 0 10px #ff6b6b; color: #ff6b6b; font-size: 18px; font-weight: bold;"); console.log("%c3D effect", "text-shadow: 1px 1px 0 #ccc, 2px 2px 0 #c9c9c9, 3px 3px 0 #bbb; font-size: 24px; font-weight: bold;");
